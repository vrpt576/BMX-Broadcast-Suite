# Roadmap

This roadmap outlines the planned phases for the BMX Broadcast Suite.

## Phase 1: Foundation
- Define core data model for BMX race events
- Document RaceManager database mapping
- Create exporter scaffolding for live data output

## Phase 2: OBS Overlay
- Build browser-source overlay templates
- Add theme support and visual layouts
- Connect live data exports to overlays

## Phase 3: Control and polish
- Add controller UI and hotkey support
- Improve theme customization and track support
- Add documentation and examples
