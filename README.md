# BMX-Broadcast-Suite
Open-source live BMX broadcasting system for OBS Studio using USABMX RaceManager.

## Features

- Live Moto display
- Live Class display
- Round display
- Rider lineups
- Lane assignments
- Theme support
- OBS Browser Source
- SQL Server integration
- USABMX RaceManager support
